HTML

Introduction HTML?

HTML is a short of "HYPERTEXT MARKUP LANGUAGE" and it is developed by Tim Burners Lee in the year 1990, by using this we can build our website.
HTML is the basic building block of the web page. This will help us to tells a web browser how to display text, images and other forms of multimedia on a webpage.

HTML Doctype:

All HTML pages must have <!DOCTYPE html> declaration.
This declaration is not a HTML Tag its just an information to the browser.

HTML Elements:

HTML element is defined by a start tag and content and an end tag Example: <h1> <p> By Siddharth </p> Some of the HTML element doesn't have end tag or they don't need it. Example: <br>,<hr>

HTML Attributes:

The HTML attribute is nothing but additional information of the tag. All in html element have attributes and it always specified in the start tag Example:image and reference tag <img src="sid profile pic.png"> <a href="www.siddharth.com">

HTML Style:

In HTML style attributes we can add styles to an element like font, size of the font, text color,... Example: <p style="background-color:red";>Siddharth</p>

BASIC TAGS IN HTML

HTML Heading:

HTML headings are titles or subtitles that we want to show on our webpage. HTML tags are define in 6 tags <h1> to <h6>. <h1> is the first priority tag and the <h6> is the lowest priority tag.

HTML paragraph:

In HTML <p> tag define as a paragraph tag. If we use the paragraph tag it automatically started on the new line. If we want a break the line without starting a new paragraph we can also use the break tag <br> inside the para tag.

HTML Formatting:

HTML text formatting is used to specialized the text. For example, if we want to highlight or bold the word we can Bold tag <b> the major text formatting tags are <b>,<strong>,<em>,<i>...

HTML Links:

<a> this is link tag. By using links we can jump from one webpage to another page Example:<a href="www.google.com">Google</a> by adding this to our website we can move to the referred website.

HTML Image:

Image tag <img> is used to add an image to our webpage Image tag also doesn't have an end tag, and this tag must have source attributes. Example:<img src="sid2867.jpeg" alt="Sid profile pic">

HTML Div:

<div> tag is known as Division tag.this tag is used to make a division in the web page and it is used to group various tags of HTML so that we can style it easily.

HTML span:

The span tag <span> is an inline container used to mark up a part of a text, or a part of a document and it is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The span tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

Inline Element and Block Element:

Block Element:

A block element always starts on a new line. Block elements cannot be nested between inline and block elements. Common block-level elements are <div>,<p>,<article>,<section>...

Inline Element:

An inline element is a part of the main element or text and not as a separate section. Inline elements can be nested between the block and inline elements. unlike block element, it does not begin on the new line Some of the inline elements are<span>,<img>,<a>...

Difference between div vs section vs article?

HTML Elements <section>,<div> and <article> are all block elements. But they have each different symantec.

<section>

The <section> element represents a generic section of a document or application. A Web site home page could be split into sections for an introduction, content, and contact information.

<div>

The <div> element is nothing but a division When no other element is suitable, the div element is used as an element of last resort. Otherwise, it leads to poor accessibility for the reader.

<article>

The <article> element represents a independent item section of content. It must have a <h1> tag for title. Like newspaper articles.

Intermediate HTML

HTML Table:

html tables allow us to create a tables contains rows and columns. in these tables <table> tag is refer to table <thead>,<tbody>and<tfoot>refers to table header,body and footer, <tr> refers to table row , <td> refers to table data.

HTML Lists:

HTML lists allow web developers to group a set of related items in lists. there are two types of lists: 1.Unordered list <ul> 2.ordered list <ol> Each list should starts with the line tag <li>. Example: <ol> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol>

HTML Form

what is HTML <form>

The HTML <form> element is used to create an HTML form for user input. Form is used to collect user input. The user input is most often sent to a server for processing.

Form Element:

Inside the <form> element can contain one or more form elements The HTML <input> element is the most used form element. Example: <input>,<label>,<textarea>,<button>...

Form Attributes:

"Action" attributes define as the what to do when the form is submitted. "Method" attributes is specifies the HTTP method to be used when submit the form data(get or post)

Input Types:

An <input> element can be displayed in many ways, depending on the type attribute. Some of the types: 1. <input type="text"> 2.<input type="email"> 3.<input type="password"> p.<input type="checkbox"> 5.<input type="button"> 6.<input type="submit">

HTML Graphics

HTML SVG:

SVG stands for Scalable Vector Graphics. <svg> has several methods for drawing box,circle,text and graphic images SVG is a language for describing 2D graphic in XML,which means we can includes a javascript event handler for each element.

HTML Canvas:

Canvas element is used to draw graphics through Javascript <canvas>s has several methods for drawing paths, boxes, circles, text, and adding images. Canvas is render by pixel by pixel.
