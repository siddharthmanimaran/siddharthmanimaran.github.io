JavaScript-JS

What is JavaScript?

JavaScript is the most popular programming language for web .
It is a scripting language and also is one of the core technologies of the World Wide Web
All major web browsers have a dedicated JavaScript engine to execute the code on the user's device.
JavaScript is one of three languages all web developers must learn
1.HTML-to define the content of the web page
2.CSS-to specify the layout of web pages
3.JavScript-to program the behavior of the web page

Data Types:

Data types is an important concept.Without data types, we cannot able to operate on variables
In Javascript, there are three data types primitive data types,composite data types and trivial data types
In primitive data types, there are three data types
1.Strings
2.Numbers
3.Booleans
In composite data types, there are two major data types
1.Array
2.Object
In trivial data types, there are two major data types
1.Null
2.Undefined

Strings

String is nothing but a "word or letters in english".
It is a series of characters like "JavaScript"
Strings are written with quotes. You can use single or double quotes
Example:var name="Siddharth Manimaran"

Numbers

Numbers in JavaScript are just numbers.
It can be written with decimals also we can use it like this:"1234" or "3.14"
Example:let num = 3.14

Booleans

Booleans are defined as true or false
Booleans are often used in conditional testing.
Example:
let x = 5;
let y = 5;
let z = 6;
(x == y) // Returns true
(x == z) // Returns false

Array

JS array is defined as a set of content
Array must be with a square brackets and it is separated by commas
Array indexes are zero-based, which means the first item is [0], second is [1], and so on.
Example:
var array = ["Rajni","kamal","Ajith","Vijay"]

Object

An object is a complex data type that allows us to store collections of data.
JavaScript objects are written with curly braces {}.
Object properties are written as name:value pairs, separated by commas.
Example:
const car = { carName:"Mahindra" carModel:"Thar" carType:"SUV" };

Null

In JavaScript, null is a special value that represents empty or unknown value.
Example:
const number = null;

Undefined

The undefined data type represents value that is not assigned.
A variable without a value, has the value undefined.
Example:
let car;
console.log(car); // its undefined

Variables

Variables are containers for storing data (values).
There are 3 ways to declare a JavaScript variable:
1.var
2.let
3.const

var

var is used in the older versions of JavaScript
var is function scoped
Example:
var name="Sid";

let

let is the new way of declaring variables starting ES6 (ES2015).
Variables defined with let cannot be Redeclare.
Variables defined with let must be Declared before use.
let is block scoped.
Some browsers that do not support let.
Example:
let name ="Sid";

const

The const keyword was introduced in ES6 (2015).
Like let,const also cannot be Redeclare and Reassigned
const is block Scope.
Example:
const num ="28071998";

Operators

In Js operators are a special symbols used to perform operations on operands.
For example:
2 = 3; //5

Types of Js operators

  1. Assignment Operators
  2. Arithmetic Operators
  3. Comparison Operators
  4. Logical Operators
  5. Bitwise Operators

Assignment Operators

Assignment operators are used to assign values to variables.
Operators Name Examples
= Assignment operator const x = 10;
+= Addition operator a += 5; //a = a + 5
-= Subtraction operator a -= 5; //a = a - 5
*= Multiplication operator a *= 5; //a = a * 5
*= Division operator a /= 5; //a = a / 5
*= Remainder operator a %= 5; //a = a % 5

Comparison Operators

Comparison operators compare two values and return a boolean value, either true or false.

Operators Description Example
== Equal to: returns true if the operands are equal x == y
!= Not equal to: returns true if the operands are not equal x != y
=== Strict equal to: true if the operands are equal and of the same type x === y
!== Strict not equal to: true if the operands are equal but of different type or not equal at all x !== y
> Greater than: true if left operand is greater than the right operand x > y
>= Greater than or equal to: true if left operand is greater than or equal to the right operand x >= y
< Less than: true if the left operand is less than the right operand x < y
<= Less than or equal to: true if the left operand is less than or equal to the right operand x <= y

Logical operators perform logical operations and return a boolean value, either true or false

Operators Description Example
&& Logical AND: true if both the operands are true, else returns false x && y
|| Logical OR: true if either of the operands is true; returns false if both are false x || y
! Logical NOT: true if the operand is false and vice-versa. !x
