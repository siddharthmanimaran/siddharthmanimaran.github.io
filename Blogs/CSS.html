CSS-Cascading Style Sheets

What is CSS?

CSS is a language for describing the presentation of Web pages CSS stands for Cascading Style Sheets. its includes colours, layout, and fonts. CSS is officially published on 12May 1998.

Ways to Write CSS:

there are 3 ways to write a CSS in out HTML document. 1.inline CSS 2.Internal CSS 3.External CSS

What is Inline CSS?

This is the old way to use the styles writing a CSS inside the HTML tags is called Inline CSS. Example: <h1 style="color: blue">siddharth</style>

What is Internal CSS?

Internal CSS is nothing but using CSS with the help of <style>tag. We can use this inside the HTML document. Example: <style> h1 { color: blue; } </style>

What is External CSS?

External CSS is use with the help of <link /> tag in the head of HTML and the file must be in .css extension Example:<head> <link link rel="stylesheet" type="text/css" href="style.css" /> </head>

CSS Selector:

CSS selector is used to find the element. Element Selector:The element selector selects HTML elements based on the element name. Example: p {color:red} 1.ID Selector:The id selector uses the id attribute of an HTML element to select a specific element. Example: #name {color:red} 2.Class Selector:The class selector selects HTML elements with a specific class attribute. Example: .name {color:red}

CSS Colors:

There are three major used color technique.

RGB:

The 3 parameter basically correponds to red,green and blue. This parameters vary from 0 to 255 Eg:RGB(0,0,255)

HEX:

Hex code starts with # Single set value can vary from 00 to 09 and AA to FF Sets basically correspond to Red, Green, and Blue Eg: #ff0000 ; means color red

RGBA:

The 4 parameter basically corresponds to red, green, blue and alpha Value of the first three parameters may vary from 0 to 255 and the last parameter ranges from 0 to 1 that is from 0.1 to 0.9 Eg: RGB(255,0,0,0)

CSS Background:

The CSS background properties are used to add background effects for elements. Color – used to set the color of the background
Image – used to set an image as the background
Position – used to determine the position of the image
Attachment – used controlling the mechanism of scrolling
Repeat – used to determine if the image has to repeat or not

CSS Box Model:

CSS Box Model is a box that warps around every HTML element and its consist of margins, borders, padding, and content. Content-The most inner part is a our content
Padding-the padding is nothing but the space between the content and the border
Border-border that goes around the padding and the content.
Margin-margin is the area outside the border.

CSS Border:

CSS Border helps in setting up the border for HTML elements There are 4 properties that can helps us to set a border: 1.width 2.style 3.color 4.radius And it can also be set in single line Eg:border: 5px solid blue;

CSS Margin:

Margin properties are used to create a space around the elements. With the help of margin we can change all four side margins 1.margin-top 2.margin-bottom 3.margin-left 4.margin-right Shorthand example:margin: 25px 50px 25px 50px

CSS Padding:

Padding is used to create space around an element content,inside of any defined borders. With the help of padding we can change all four side of the content inside the border. 1.padding-top 2.padding-right 3.padding-bottom 4.padding-left Shorthand Example: padding:25px 50px 25px 50px

CSS Width and Height:

Height and Width is used to set the height and width of the element content. With the help of these properties wwe can change the content width and height. Example:width:200px; height:100px;

CSS Flex Box:

Flex Box is nothing but Flexible Box Module And its a efficent way to layout,align and distribute space First we need to define flex container .

Flex Container:

The flex container properties are flex-direction, flex-wrap,flex-flow, justify-content, align-items and align-content. display: its a type of box used for an HTML element. flex-direction: The direction of the flexible items inside a flex container flex-warp: The flex items should wrap or not. flex-flow: A shorthand property for flex-direction and flex-warp. align-items: it will align the flex item vertically align-content: It is similar to align-item, but instead of aligning flex items, it aligns flex lines

Flex Item:

The child element of the container is called as flex item. The flex item properties are order,flex-grow,flex-shrink,flex-basis,flex and align-self order: Its a order of the flex items inside the same container. align-self: Specifies the alignment for a flex item flex: A shorthand property for the flex-grow, flex-shrink, and the flex-basis properties flex-basis: Specifies the initial length of a flex item flex-grow: Specifies how much a flex item will grow relative to the rest of the flex items inside the same container flex-shrink: Specifies how much a flex item will shrink relative to the rest of the flex items inside the same container
